# Math Battle

Math Battle is an educational multiplayer game project written in **Go**. The primary goal of this repository is to practice backend development, networking, and real-time multiplayer architecture.

> 🚧 This repository is a learning project focused on improving Go backend skills.

## ✨ What I practiced

* HTTP API
* WebSocket communication
* Multiplayer room management
* Concurrent programming (goroutines, channels, mutexes)
* Game loop architecture
* JSON serialization
* Real-time client/server synchronization

## 🎮 Game Modes

### 🌋 Lava Pillars

A multiplayer game where players must survive by jumping across lava while the arena changes over time.

**Status:** Playable.

### 📈 Formula Wars *(In Development 🚧)*

Formula Wars is currently under active development and is **not available yet**.

The gameplay is being redesigned, so this mode is temporarily disabled.

## 🛠 Tech Stack

* Go
* Gin
* Gorilla WebSocket
* JSON
* Goroutines
* Mutexes

## 📚 Purpose

The purpose of this repository is to learn how to build multiplayer backend systems in Go.

This project helped me practice:

* WebSocket servers
* Real-time synchronization
* Room management
* Game state updates
* Clean project organization
* Backend architecture

## 🚀 Future

Development of **Lava Pillars** is complete enough for experimentation.

**Formula Wars** is still being redesigned and may change significantly before becoming playable.

This repository will continue to serve as a place to experiment with multiplayer game mechanics and backend technologies in Go.
